# Php5.6 Migrations
This development has been carried out mainly for a symfony 1.4 legacy application with a Propel version without a migration system.

## Configuration

Complete the following information in a `.php` in `Configuration/` path.
```
<?php
return [
    'ENGINE' => '',
    'USERNAME' => '',
    'PASSWORD' => '',
    'HOST' => '',
    'DATABASE' => '',
    'PORT' => ''
];

```
